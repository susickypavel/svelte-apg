## 2. Local development

### 2.1 Prerequisites

- [**Node >=20**](https://nodejs.org/en)
- [**pnpm >=8**](https://pnpm.io)

### 2.2 Installation

```sh
git clone https://github.com/susickypavel/svelte-apg.git

cd svelte-apg

pnpm install

# cd to project you want as mentioned above (0. Projects)
# follow README.md in project folder
```
